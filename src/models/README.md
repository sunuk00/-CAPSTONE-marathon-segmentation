# Extracting Path Pixels from Mask Images
마스크 이미지에서 경로 픽셀의 두께를 1px로 줄이는 과정이 필요하다. 왜냐하면 모델이 출력한 마스킹 이미지를 실제 (위도,경도)로 변환할 때, 픽셀의 중심 좌표를 사용하기 때문이다. 따라서 경로가 너무 두꺼우면 모델이 예측한 마스킹 이미지에서 픽셀 중심 좌표가 실제 경로와 멀어질 수 있다. 이를 방지하기 위해, 마스크 이미지에서 경로 픽셀의 두께를 1px로 줄이는 작업이 필요하다. 이렇게 하면 모델이 예측한 마스킹 이미지에서 픽셀 중심 좌표가 실제 경로에 더 가깝게 된다

알고리즘의 순서는 다음과 같다:
1. 마스크 이미지를 이진 배열로 읽는다.
2. 작은 성분을 제거한다.
3. Zhang-Suen thinning으로 skeleton을 만든다.
4. skeleton을 그래프로 변환한다.
5. 그래프에서 가장 긴 경로를 찾는다.
6. 가장 긴 경로의 좌표를 (y, x) 리스트로 반환한다.
7. (y, x) 좌표를 (x, y)로 바꿔서 반환한다.
